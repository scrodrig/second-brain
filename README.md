# Second brain

This project is addressed to build a solution for a **Second Brain**, a quite popular template for Notion.

[Second brain template](https://www.notion.so/templates/second-brain)

## Installation

Install `second-brain` with npm

```bash
  cd second-brain
  npm install
```
